Web 前端的三件套是 HTML、CSS 和 JavaScript。HTML 负责结构，CSS 负责样式，JavaScript 负责交互。你可以把 HTML 理解成文章骨架，把 CSS 理解成排版和视觉规则，把 JavaScript 理解成让页面对用户操作作出反应的脚本。

一个最小 HTML 页面可以包含标题、段落、链接和列表。HTML 标签不是为了“画界面”，而是给内容赋予语义。例如 `h1` 表示一级标题，`p` 表示段落，`a` 表示链接，`ul` 和 `li` 表示列表。语义清楚的 HTML 更容易被浏览器、搜索引擎和辅助工具理解。

CSS 负责控制颜色、字号、间距、布局和响应式适配。初学者不必一开始学习复杂动画，更应该先掌握盒模型、flex 布局、grid 布局和媒体查询。网页看起来混乱，很多时候不是审美问题，而是间距、对齐和层级没有处理好。

JavaScript 让网页可以交互。例如点击按钮后展开内容，输入关键词后过滤列表，向接口请求数据并更新页面。学习 JavaScript 时，要先理解变量、函数、事件和 DOM。DOM 是浏览器把 HTML 解析后形成的结构，JavaScript 可以读取和修改它。

静态网站是最适合初学者展示成果的方式。你可以写一个个人学习页面，列出课程、项目、笔记和联系方式，然后部署到 GitHub Pages。这个过程会串起 HTML、CSS、路径、Git、构建和部署，比单独看教程更有效。

本章练习：创建一个 `index.html`，包含你的名字、学习目标、三条最近完成的笔记链接。再创建一个 `style.css`，设置最大宽度、字体、行距和链接颜色。最后加一个简单按钮，用 JavaScript 点击后显示一句学习格言。这个页面不需要华丽，但要清楚、可读、可访问。

常见误区是过早追求复杂框架。React、Vue、Astro 都很有用，但如果你完全不懂 HTML、CSS 和 JavaScript，框架只会让问题更抽象。先理解浏览器原生工作方式，再学习框架，路线会稳很多。
